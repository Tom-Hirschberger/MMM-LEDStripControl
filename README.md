# MMM-LEDStripControl ##
This module reads temperature of different sensors (default HTU21D) with external scripts and displays the values.

## Screenshots ##

## Installation ##
### Module ###
```
	cd ~/MagicMirror/modules
    git clone https://github.com/Tom-Hirschberger/MMM-LEDStripControl.git
    cd MMM-LEDStripControl
    npm install
```

## Configuration ##

```json5
        {
            module: "MMM-LEDStripControl",
            position: "middle_center",
            config: {
            },
        },
```
